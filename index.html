My First PWA

This is an example of how to convert a website into a PWA.

REQUIREMENTS PWA CONVERT

1. Add the following tags on the application home page

// Paint the browser's address bar

meta name = "theme-color" content = "#1a237e">

// Tell mobile browsers that this site is optimized for mobile (It's friendly: HandheldFriendly)

meta name = "MobileOptimized" content = "width">

meta name = "HandheldFriendly" content = "true">

// Meta tags for Apple devices

meta name = "apple-mobile-web-app-capable" content = "yes">

meta name = "apple-mobile-web-app-status-bar-style" content = "black-translucent">

// Background image to be seen when executing/loading the PWA installed on the device

link rel = "apple-touch-startup-image" href = "./img/startup_image_wp.jpeg">

// Icon that is installed directly on the screen of Apple devices

link rel = "apple-touch-icon" href = "./img/icon_16.png">

// Favicon

link rel = "shortcut icon" type = "image/png" href = "./img/ icon_16.png">

// Manifest.json

link rel = "manifest" href = "./manifest.json">

2. Create the file manifest.json in the root of the website

3. Create two javascript files

call_sw.js => Script that calls the Service Worker. This links to the home page of the website

sw.js => The Service Worker
